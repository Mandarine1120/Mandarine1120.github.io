# 英语学习

# 单词

# a

  • awfully：极度的
  • auxiliary：辅助的，备用的
  • ambiguous：不明确的

# b

  • bribe：贿赂
  • bluffing：虚张声势的

# c

  • compensation：补偿
  • circumstances：条件
  • contrastive：对比的

# d

  • dismissed：解散
  • decouple：解耦

# h

  • hick：乡巴佬

# i

  • initiative：倡议，主动性
  • intra-class and inter-class：类内与类间

# k

  • kaleidoscopic：万花筒般的

# m

  • migrate：迁徙，移民

# p

  • presumptuous：专横的；放肆的；冒昧的；自负的
  • punter：顾客
  • plug-and-play：即插即用

# r

  • ridiculous：荒谬的

# s

  • squalor：悲惨的，肮脏的
  • screwed：拧紧的
  • strides：大步走
  • skeleton：骨骼

# t

  • talent：有天赋的

# u

  • ultimate：最好的，最终的，最后的

# w

  • weird：奇怪的
  • wire：金属丝，导线；wired：联网的
  • worrywart：爱操心的人
  • whooposie：哎呀呀

# 短语

# g

  • Give me a favor：帮我个忙
  • Get over yourself：别自作多情了

# i

  • I have a confession to make：我要坦白一件事

# o

  • On it：收到 / 这就去

# w

  • With all due respect：恕我直言，容我冒昧
